## For Users

### Getting Started

1. **Visit the Website**
   - Go to [your-domain.com] or open landing.html
   - No signup required!

2. **Create Your Cat's Profile**
   - Click "Add New Cat"
   - Enter name, weight, age, activity level
   - Save (stored locally on your device)

3. **Calculate Daily Portions**
   - Click "Calculate Daily Needs"
   - See PMR 80/10/10 breakdown
   - Save or screenshot for reference

4. **Plan Your Transition**
   - Go to Transition Planner
   - Select your cat
   - Choose current diet and speed
   - Generate customized plan

5. **Track Progress**
   - Use Health Log to record observations
   - Note stool quality, coat, energy
   - Watch for trends over time

6. **Share & Connect**
   - Share your plan to Instagram
   - Tag @anymothings
   - Join the community
   - Ask questions via DM

## Hosting Options

### Option 1: GitHub Pages (Free, Easiest)

1. Create GitHub account
2. Create new repository: `anymothings-pmr`
3. Upload all files
4. Go to Settings > Pages
5. Enable GitHub Pages
6. Your site: `username.github.io/anymothings-pmr`

**Pros:** Free, easy, automatic updates
**Cons:** URL not custom, public repository

### Option 2: Netlify (Free, Better)

1. Sign up at netlify.com
2. Drag & drop your folder
3. Get instant URL: `anymothings.netlify.app`
4. Optional: Connect custom domain

**Pros:** Free, custom domain, HTTPS, fast
**Cons:** None really!

### Option 3: Custom Domain (Professional)

1. Buy domain: `anymothings.com` or `anymothings.hk`
2. Use Netlify/Vercel/GitHub Pages
3. Connect domain in settings
4. Add SSL certificate (usually automatic)

**Cost:** HK$100-200/year for domain
**Benefit:** Professional, memorable, brandable

## Common Questions & Answers

### "Do I need to know coding?"
No! Just upload files and share the link. Everything works as-is.

### "Can users' data be lost?"
Data is stored locally in browser. Users should:
- Not clear browser data
- Use same device/browser
- Consider exporting data (future feature)

### "What if someone has questions?"
Direct them to:
1. Vet FAQ page
2. Instagram DM
3. Disclaimer page for legal questions

### "How do I update content?"
1. Edit HTML files
2. Re-upload to hosting
3. Changes appear immediately

### "Can I translate to other languages?"
Yes! Duplicate pages and translate text. Keep structure same.

### "What about iOS app?"
PWA works like an app on iOS. No App Store needed!
